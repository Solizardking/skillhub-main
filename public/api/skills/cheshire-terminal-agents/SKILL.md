# Cheshire Terminal Agents (one-shot pack)

Install **everything** in this pack with a single Skill Hub command — same idea as:

```bash
npm i cheshire-terminal-agents
```

## One-shot install

```bash
# Skill Hub (copies every member skill into your agent skills root)
npx github:Solizardking/skills install cheshire-terminal-agents --force

# npm package (SDK + CLI + skills tree under node_modules)
npm i cheshire-terminal-agents
```

Aliases: `cheshire-agents`, `cheshire-robinhood-agents` (installer expands to this pack).
